# Mapping MLC Links to IRACplus Sections

| MLC Link | IRACplus Section | Explanation |
| --- | --- | --- |
| **1. Source of Authority** | Rule | Identify the statutes, constitutions or regulations granting authority.  Include citations in the Rule section. |
| **2. Jurisdictional Scope** | Issue / Analysis | Frame the issue to specify the relevant jurisdiction; in analysis, verify that the facts fall within scope. |
| **3. Role Conferral** | Analysis | Discuss whether the actor holds the necessary role (e.g., licensed professional, authorised agent). |
| **4. Normative Content** | Rule | Set out the obligations, permissions or powers applicable to the role. |
| **5. Authorised Process** | Analysis | Explain whether the actor followed required procedures; highlight procedural requirements. |
| **6. Legal Event** | Facts / Analysis | Describe the legally significant act; in analysis, confirm it was produced by an authorised process. |
| **7. Recognition / Recording** | Analysis | Determine whether the event was formally recognised (e.g., filed, proclaimed). |
| **8. Remedy / Repair** | Conclusion / Structural Diagnosis | Address available remedies; if the chain fails, recommend repairs. |

By aligning each MLC link with the IRACplus sections, legal writers systematically address structural issues alongside doctrinal analysis.
